# Gidrolock Modbus Scanner
Простой Modbus сканнер, заточенный под устройства Gidrolock.

Кнопка "Подключиться" отправляет запрос на чтение Holding Register'а по адресу `128` (значение Modbus ID для устройств Gidrolock). 

Сообщает об ошибке после двух секунд ожидания ответа.
